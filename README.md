# WhatsApp Screenshot Sender 🤖📸

Um projeto de automação em Python que captura screenshots de sites e envia via WhatsApp, com interface gráfica amigável e operação em segundo plano.

## 📌 Funcionalidades

- 🌐 Acessa qualquer site via Selenium
- 📸 Captura screenshot automaticamente
- 📱 Envia a imagem para contatos do WhatsApp
- 🖥️ Interface gráfica com PyQt5
- 🕶️ Opção para rodar em segundo plano
- 📁 Gerenciamento automático de drivers com webdriver-manager

## ⚙️ Pré-requisitos

- Python 3.8+
- Google Chrome instalado
- Conta no WhatsApp Web ativa

## 🛠️ Instalação

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/whatsapp-screenshot-sender.git
cd whatsapp-screenshot-sender
```
## ⚠️ Limitações
- Requer que o WhatsApp Web já esteja logado
- O Chrome deve estar instalado
